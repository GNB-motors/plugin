# Icon Placeholders

Place your extension icons in this directory:

- `icon16.png` - 16x16 pixels (toolbar icon)
- `icon48.png` - 48x48 pixels (extension management)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

You can generate these from a single SVG or high-res PNG using online tools like:
- https://www.favicon-generator.org/
- https://realfavicongenerator.net/

**For now, the extension will work without icons, but you'll see broken image placeholders in the browser.**

Recommended colors for FleetEdge branding:
- Primary: #1e40af (blue)
- Accent: #22c55e (green)
- Background: #ffffff (white)
